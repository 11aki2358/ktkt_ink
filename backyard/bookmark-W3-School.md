##  W3 School

めっちゃ丁寧なhtml, CSS, javaScript講座。
オール英語だけれど、一つ一つのチャプターはとても小さいから、流し見しているだけでもなんとなく内容がわかる。
実際にコードを書く&結果を見ることが出来るのでgood。
各チャプターで色んな変数とかを網羅して紹介しているから、「こんな機能・設定があったんだ～」と面白い。

* <a href="https://www.w3schools.com/html/html_attributes.asp" target="_blank">HTML Attributes</a>  
  The title attributeのところ
* <a href="https://www.w3schools.com/html/html_formatting.asp" target="_blank">HTML Formatting</a>  
  文字の装飾。いろいろある。
* <a href="https://www.w3schools.com/html/html_links.asp" target="_blank">HTML Links</a>  
  Button as a Link, Link Titles
  * <a href="https://www.w3schools.com/html/html_links_colors.asp" target="_blank">Link Colors</a>  
    a::hoverとかa::visitedとか
* <a href="https://www.w3schools.com/html/html_images.asp" target="_blank">HTML Images</a>
  * <a href="https://www.w3schools.com/html/html_images_imagemap.asp" target="_blank">Image Map</a>  
    画像上の特定の位置・座標を指定(リンクを貼るなど出来る)
  * <a href="https://www.w3schools.com/html/html_images_picture.asp" target="_blank">The Picture Element</a>  
    枠の幅などによって、表示する画像を切り替えられる(レスポンシブ対応など)
* <a href="https://www.w3schools.com/html/html_lists.asp" target="_blank">HTML Lists</a>  
  * <a href="https://www.w3schools.com/html/html_lists_unordered.asp" target="_blank">Unordered Lists</a>  
    丸ぽちの形いろいろとか、Horizontal List with CSS(要素を横に並べる)とか
  * <a href="https://www.w3schools.com/html/html_lists_ordered.asp" target="_blank">Ordered Lists</a>  
    数字の書き方とか、1以外からリストし始めるやり方とか
* <a href="https://www.w3schools.com/html/html_responsive.asp" target="_blank">HTML Responsive</a>  
  レスポンシブの基礎。忘れないように。
* <a href="https://www.w3schools.com/html/html_computercode_elements.asp" target="_blank">HTML Computercode</a>  
  コードのほか、キーボードとか変数とかの書き方も
* <a href="https://www.w3schools.com/html/html_layout.asp" target="_blank">HTML Layout</a> / <a href="https://www.w3schools.com/html/html5_semantic_elements.asp" target="_blank">HTML Semantics</a>  
  ページ構成についての基礎。これもちゃんと押さえておきたい。
