Grid Layout

作りたかったもの
  • header : 常にウィンドウ最上部に固定して表示
  • main : headerとfooterの間に、隙間なく配置
  • footer :
    • コンテンツが多い場合 : ページ最下部に表示
    • コンテンツが少ない場合 : ウィンドウ最下部に固定
headerを最上部に固定するのは 

        header{
          position: fixed;
        }
で、 footerをページ最下部(mainが少ないときには、ウィンドウの最下部)に固定するのは 

        footer{
          position: absolute; 
          bottom: 0;
        }
みたいな感じで実装できる。
mainの高さは、「mainのコンテンツがぴったりはまるように」決まる。
だが、mainの量が少なくてウィンドウをスクロールする必要がないとき、mainの底辺とfooterの上辺との間に空白ができてしまう。
この、mainとfooterの間の空白を埋める方法を探した。 marginやpaddingの設定では、完全にはならない(無駄な余白が出来たり、コンテンツがはみ出たり)。
これの解決で役に立ったのが、CSSのgridレイアウト。
出来上がった構成は、ここ(別窓で開く)で確認すると分かりやすい。
htmlの構成は
  • html
    • body
      • header
      • main
      • footer
bodyにgridを適用し、その子要素header, main, footerを縦に3つ並べる。 最低限必要なCSSは、以下の通り。 

        html {
          height: 100%;
        }
        
        body {
          background-color: wheat;
          margin: 0;
          height: 100%;
          display: grid;
          grid-template-rows: 12vw auto 10vh;
          grid-template-columns: 100%;
          grid-auto-flow: column;
        }
        
        body header {
          grid-row: 1 /3;
          grid-column: 1/1;
          background-color: aquamarine;
          width: 100%;
          position: fixed;
          height: 12vw;
        }
        
        body main {
          grid-row: 2/3;
          grid-column: 1/1;
          background-color: violet;
          width: 90%;
          margin: auto;
          height: 100%;
        }
        
        body footer {
          grid-row: 3/3;
          grid-column: 1/1;
          background-color:tan;
          bottom: 0;
          height: 10vh;
        }
headerの高さ 12vwとfooterの高さ10vhは、好きに変えていい。
