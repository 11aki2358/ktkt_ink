コードの折り返し

preタグでくくったコード(ブロック要素)が横長になった場合には、スクロールを付ける。
本文中のコード(インライン要素)が長くなった場合は、適当なところで改行したい。

  • prism.cssの編集
    
  code[class*="language-"],
  pre[class*="language-"] {
    (中略)
    /* white-space: pre; */
  }

  • style.cssの編集

    
  code{
    white-space: pre-wrap;
  }

  pre code{
    white-space: pre;
  }
